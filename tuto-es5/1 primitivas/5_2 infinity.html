¿Cuanto es infinito negativo mas infinito positivo? => Es NaN?

Infinity es una del objeto global, es decir, esta es una variable de alcance global.

  • El valor inicial de Infinity es Number.POSITIVE_INFINITY.
  • El valor de Infinity (infinito positivo) es mayor que cualquier otro numero incluido el mismo.
  • Este valor se comporta matematicamente como el infinito matemático;
  • por ejemplo, cualquier numero multiplicado por Infinity da como resultado Infinity,
  • y cualquier numero divido por Infinity da como resultado cero.

    • Para la especificación ECMAScript 5, Infinity es de solo lectura (implementado en JavaScript 1.8.5 / Firefox 4).

    var a = Infinity;
    var b = Number.NEGATIVE_INFINITY;
    console.log("numero a " + a);
    console.log("numero b " + b);
    console.log("a > a ? " + (a>b)); // true
    console.log("a + b => " + (a+b)); // Nan
    console.log("1 * a => " + (1*a)); // Infinity
    console.log("1 * b => " + (1*b)); // -Infinity
    console.log("1 / a => " + (1/a)); // 0
    console.log("1 / b => " + (1/b)); // 0 , nota: esperaba q fuera -0 en realidad

    Toma en cuentas las propiedades:

    Number.NEGATIVE_INFINITY

    Number.POSITIVE_INFINITY
